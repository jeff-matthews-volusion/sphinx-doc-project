Build Swagger API ReferenceÂ¶

To build Swagger API reference:

  1. Clone the swagger git repo

  2. Clone the swagger-ui git repo

  3. Navigate to local swagger repo

    1. npm install
    2. gulp

  4. Edit index.html in swagger-uisrcmainhtml and add “material.api.json” to the URL location around line 54

  5. Copy material.api.json from swaggerdist to swagger-uisrcmainhtml

  6. Navigate to local swagger-ui repo

    1. npm install
    2. gulp
    3. gulp serve
    4. Ctrl+C to stop server
    5. Navigate to localhost:8080 in browser
